# 1 - Calculator

## Description

- Console app
- Basic operations (addition, subtraction, multiplication, division)
- Input validation
- Error handling
- Repeated usage without restarting the program

## Example

Введите 1-ое число:
5
Выберите операцию:
1. + (сложение)
2. - (вычитание)
3. * (умножение)
4. / (деление)
1
Введите 2-ое число:
3
Результат: 8
